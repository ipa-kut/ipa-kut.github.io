Detailed Description

template<typename T>
struct ambs_base::AMBSPort< T >

Struct that is used to represent one port in the interfaces of components.

Characterised by topic name, a buffer copy of the latest messgae received on the topic if subscriber, an index to identify its position in the list of all interface ports, as well as a subscriber and a publisher associated with that topic. Currently the ports are initialised ot have a subscriber OR publisher, not both.

All interaction with the port (reading values, publishing values) is to be done using this object only
